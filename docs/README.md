# FoodBridge Demo Pages

This directory contains the static demo pages for FoodBridge, deployed to GitHub Pages.

## Live Demo

Visit the live demo at: `https://engdawood.github.io/FoodBridge/`

## What's Included

- **index.html** - Main landing page showcasing the FoodBridge platform
- Features overview
- Technology stack information
- User roles and capabilities
- Demo account credentials

## Local Development

To view the pages locally:

1. Open `index.html` in your web browser
2. Or use a local server:
   ```bash
   python -m http.server 8080
   # Then visit http://localhost:8080
   ```

## Deployment

The pages are automatically deployed to GitHub Pages via GitHub Actions whenever changes are pushed to the `main` branch in the `docs/` directory.

## Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
